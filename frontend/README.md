# Frontend - Vite React App

This project uses [Vite](https://vitejs.dev/) for a fast React development experience.

## Available Scripts

In the `frontend` directory, you can run:

### `npm run dev`

Runs the app in development mode using Vite.  
Open [http://localhost:3000](http://localhost:3000) to view it in your browser.

The page will reload when you make changes.  
You may also see any lint errors in the console.

### `npm run test`

Runs the test suite using [Vitest](https://vitest.dev/).  
Test files are located alongside components or in the `src` directory.

### `npm run build`

Builds the app for production to the `dist` folder.  
It correctly bundles React in production mode and optimizes the build for the best performance.

### `npm run preview`

Serves the production build locally for previewing.

## Environment Variables

- See `.env.example` for required variables (e.g., `VITE_BACKEND_URL`).
- The app uses `VITE_BACKEND_URL` to communicate with the backend API.
